Project 1

Online Proctor System

Proctor system is a system which helps academic institutes and their faculties to manage large amount of student data like personal information, academic progress, student's co-curricular and extra-curricular activities and access it easily whenever needed. It is a web-based application which uses HTML, CSS, Bootstrap and JavaScript for front-end. All data is stored securely on SQL servers which are managed by the college administrator and ensures highest possible level of security. With the help of my team members - Harsh Churi, Anish Dalvi and Shravani Gavali we have created this as our college mini project.
